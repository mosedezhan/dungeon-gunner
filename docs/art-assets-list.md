# 全项目美术素材清单

> 本文档为美术素材制作的完整交付规格。每个素材包含精确的技术参数。

## 通用技术规范

| 项目 | 规范 |
|------|------|
| **文件格式** | PNG，透明背景（alpha = 0 为透明区域） |
| **动画图片类型** | 水平序列帧雪碧图（frames 从左到右排列） |
| **静态图片类型** | 单张 PNG |
| **角色朝向** | 详见下方"朝向规则"小节 |
| **视角** | 顶视角略带 3/4 俯视（能看到角色面部五官） |
| **艺术风格** | D&D 西幻世界观 + 美式卡通像素画风（参考 Enter the Gungeon）。粗黑轮廓线，夸张比例 |
| **调色板** | 不限色数，各素材间保持风格统一即可 |
| **雪碧图排列示例** | 4 帧 48×48 的雪碧图，总尺寸为 192×48：`[帧0][帧1][帧2][帧3]` |

### 朝向规则

角色/敌人的移动类动画（idle、run、walk）需要 **2 个朝向**：

```
_side（侧视朝右）          _down（正朝下）

      ← 角色面朝右           ↓ 角色面朝屏幕下方
        ┌──┐                  ┌──┐
        │😊│ ← 侧脸           │😊│ ← 正脸
        └──┘                  └──┘

运行时方向映射:
  向右 → 播放 _side 原图
  向左 → 播放 _side 水平翻转
  向下 → 播放 _down 原图
  向上 → 播放 _down 垂直翻转
```

**不需要方向区分的动画**：hurt（受击）、death（死亡）— 只需 1 套，不做 `_side` / `_down` 区分。

**不需要方向区分的素材**：武器、弹丸、特效、拾取物、UI 卡片 — 均为单朝向或全向（圆形/放射状）。
