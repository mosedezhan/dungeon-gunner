## 依赖关系

```
角色系统 ← 移动与输入 ← 战士攻击/法师攻击
                      ↘ 敌人系统 ← 波次系统 ← 精英敌人/宝箱怪
经验与升级 ← 卡牌系统
场景流转（独立）
序列帧动画（贯穿所有视觉表现）
```

## 设计顺序

基础 → 核心 → 功能 → 表现：

1. 角色系统 → 2. 移动与输入 → 3-4. 攻击系统 → 5. 敌人系统 → 6. 波次系统 → 7. 经验升级 → 8. 卡牌系统 → 9-11. 功能系统 → 12-13. 表现系统
