# 掉落物系统 GDD

## 1. 概述

定义击杀敌人后产生的掉落物——经验球和血球。经验球提供升级所需经验值，血球提供非卡牌来源的 HP 恢复，作为防止无回血卡牌时零回复的安全网。两种掉落物共用拾取机制，简化实现。

## 2. 玩家幻想

击杀敌人后看到绿色经验球散落一地，偶尔混着一颗红色血球——在血量紧张时看到红血球是一种如释重负的感觉。路过自动拾取，不需要刻意操作。

## 3. 详细规则

### 掉落物种类

| 类型 | 颜色 | 效果 | 来源 |
|------|------|------|------|
| 经验球 | 绿色 | 获得 xpReward 值经验 | 所有敌人死亡 |
| 经验球（稀有） | 蓝色 | 获得 xpReward ×3 经验 | 精英敌人死亡 |
| 血球 | 红色 | 回复 20 HP | 10% 概率掉落 |

### 经验球掉落

- 敌人死亡时在其位置生成 1 个经验球
- 精英敌人死亡时生成 3 个经验球（每个为基础 xpReward 值）
- 经验球在死亡位置随机散射（半径 20-40px）

### 血球掉落

- 每个敌人死亡时独立判定：10% 概率掉落 1 个血球
- 精英敌人掉落概率：25%
- 血球在死亡位置随机散射（同经验球）
- 血球回复 20 HP
- 玩家满 HP 时仍可拾取血球（回复量不超过 maxHp）

### 通用拾取机制

| 属性 | 值 |
|------|-----|
| 拾取半径 | 20px（大于视觉半径，便于拾取） |
| 存在时间 | 无限（不会消失） |
| 拾取方式 | 靠近自动拾取 |

### 掉落物合并（性能优化）

- 同一区域多个经验球合并为一个大球，显示总经验值
- 血球不合并（需要独立计数）
- 超过 50 个未拾取掉落物时，最远的自动飞向玩家（磁吸清理）

## 4. 公式

### 血球掉落判定
```
onEnemyDeath(enemy):
    // 经验球
    xpCount = enemy.isElite ? 3 : 1
    for i = 0 to xpCount:
        spawnXpOrb(enemy.position + randomScatter(20, 40))

    // 血球
    dropChance = enemy.isElite ? 0.25 : 0.10
    if random() < dropChance:
        spawnHealthOrb(enemy.position + randomScatter(20, 40))
```

### 掉落物散射
```
scatterOffset = randomUnitVector() × random(20, 40)
spawnPosition = deathPosition + scatterOffset
```

### 血球回复
```
actualHeal = min(20, player.maxHp - player.hp)
player.hp += actualHeal
```

## 5. 边缘情况

- **满 HP 拾取血球**：正常拾取，回复量不超过 maxHp（不浪费，但也不超额）
- **同时拾取多个掉落物**：各自独立结算，经验溢出保留到下一级
- **拾取血球时触发升级**：先回复 HP，再处理升级流程
- **掉落物过多性能问题**：超过 50 个时触发磁吸清理
- **经验球合并后拾取**：一次性获得合并后的总经验
- **玩家死亡后掉落物**：掉落物保留在场景中，但不影响 GameOver 流程

## 6. 依赖

### 上游依赖
- **敌人系统**：提供 xpReward、isElite、死亡位置
- **角色系统**：需要玩家位置、当前 HP、maxHp

### 下游依赖
- **经验与升级**：经验球拾取增加经验值，触发升级
- **HUD**：显示经验条、HP 条变化

## 7. 可调参数

| 参数 | 默认值 | 安全范围 | 影响方面 |
|------|--------|---------|---------|
| 血球掉落概率（普通） | 10% | 5-20 | 生存安全网 |
| 血球掉落概率（精英） | 25% | 10-40 | 精英击杀奖励 |
| 血球回复量 | 20 HP | 10-40 | 单次回复价值 |
| 经验球拾取半径 | 20px | 10-50 | 拾取便利性 |
| 经验球散射半径 | 20-40px | 10-80 | 掉落分散度 |
| 磁吸清理阈值 | 50 个 | 30-100 | 性能保护 |

- [ ] 普通敌人 10% 概率掉落血球（红色）
- [ ] 精英敌人 25% 概率掉落血球
- [ ] 血球回复 20 HP，不超过 maxHp
- [ ] 玩家靠近 20px 内自动拾取所有掉落物
- [ ] 掉落物在死亡位置随机散射 20-40px
- [ ] 超过 50 个未拾取掉落物时触发磁吸清理
