# 卡牌系统 GDD

## 1. 概述

定义升级时出现的卡牌品质、效果和随机机制。玩家每次升级从 3 张随机卡牌中选择一张，卡牌按品质分为 4 个等级，品质越高效果越强甚至改变玩法。

## 2. 玩家幻想

每次升级都是一次期待感——"这次会出什么卡？"普通卡稳步提升，稀有卡带来明显提升，史诗卡让玩法产生变化，传说卡改变整个构建方向。选卡的过程就是塑造角色成长方向的过程。

## 3. 详细规则

### 品质分级

| 品质 | 颜色 | 出现概率 | 定位 |
|------|------|---------|------|
| 普通 | 白色 | 60% | 基础数值提升 |
| 稀有 | 蓝色 | 25% | 较强数值提升 |
| 史诗 | 紫色 | 12% | 强力效果/复合提升 |
| 传说 | 橙色 | 3% | 改变玩法的效果 |

### 卡牌池

#### 通用卡（两个职业共享）

| 卡牌名 | 品质 | 效果 | 值 |
|--------|------|------|-----|
| 生命强化 | 普通 | maxHp + | 20 |
| 移速强化 | 普通 | moveSpeed + | 15 |
| 恢复 | 普通 | 回复 HP | 30 |
| 强壮 | 稀有 | maxHp +, 回复 | +40, 回满 |
| 疾步 | 稀有 | moveSpeed + | 30 |
| 再生 | 史诗 | 每秒回复 HP | 2 |
| 不死 | 传说 | 受致命伤时保留 1HP（每局 1 次） | — |

#### 战士专属卡

| 卡牌名 | 品质 | 效果 | 值 |
|--------|------|------|-----|
| 力量 | 普通 | damage + | 5 |
| 猛击 | 稀有 | damage +, attackAngle + | +8, +20° |
| 旋风 | 史诗 | attackAngle + | 60° |
| 狂战士 | 传说 | HP 越低 damage 越高（每损失 10% HP → damage ×1.1） | — |

#### 法师专属卡

| 卡牌名 | 品质 | 效果 | 值 |
|--------|------|------|-----|
| 魔力 | 普通 | damage + | 3 |
| 连射 | 稀有 | fireRateMs - | -40ms |
| 穿透 | 史诗 | 子弹穿透 1 个敌人 | — |
| 弹幕 | 传说 | 每次攻击发射 3 颗子弹（扇形） | 角度 30° |

### 选卡流程
1. 升级触发时生成 3 张卡牌
2. 每张卡牌独立按概率抽取品质，再从对应池中随机选择
3. 3 张卡牌不重复（同名不重复出现）
4. 卡牌至少包含 1 张当前职业专属卡
5. 选择后效果立即生效

### 品质保底
- 连续 5 次未出稀有或以上品质，第 6 次保证至少 1 张稀有
- 连续 15 次未出史诗或以上品质，保证至少 1 张史诗
- 传说卡无保底

### 数值叠加
- 同名卡牌可重复获得（除传说卡外）
- 叠加效果为线性累加（damage +5 再 damage +5 = damage +10）
- 传说卡每张全局唯一，获得后从池中移除

## 4. 公式

### 品质抽取
```
roll = random(0, 100)
if roll < 3:        → 传说 (3%)
else if roll < 15:  → 史诗 (12%)
else if roll < 40:  → 稀有 (25%)
else:               → 普通 (60%)

3 张卡牌各自独立抽取
```

### 狂战士传说卡伤害计算
```
actualDamage = baseDamage × (1 + 0.1 × floor(missingHpPercent / 10))

变量定义：
- baseDamage: 当前基础伤害（含其他卡牌加成）
- missingHpPercent: (maxHp - currentHp) / maxHp × 100

示例：
- 满血: damage × 1.0
- 50% HP: damage × 1.5
- 10% HP: damage × 1.9
```

### 弹幕传说卡
```
bulletCount = 3
spreadAngle = 30°  (总扇形角度)
每颗子弹角度: -15°, 0°, +15°（相对于鼠标方向）
每颗子弹伤害 = baseDamage（不衰减）
```

## 5. 边缘情况

- **卡牌池抽空**：传说卡抽完后不再出现，其余品质卡牌池足够大不会抽空
- **同名卡多次获得**：线性叠加，无上限（如多次获得"力量"卡伤害持续增长）
- **升级时所有职业卡都不适用**：不会发生，至少保证 1 张职业专属卡
- **传说卡"不死"触发后**：该卡效果消耗，后续不再生效，但不从已获得列表移除
- **弹幕 + 穿透叠加**：3 颗子弹各自独立穿透，最多各穿透 1 个敌人

## 6. 依赖

### 上游依赖
- **经验与升级**：升级时触发选卡
- **角色系统**：需要职业类型决定专属卡池

### 下游依赖
- **战士攻击 / 法师攻击**：卡牌效果修改攻击参数
- **HUD**：显示已获得的卡牌列表

## 7. 可调参数

| 参数 | 默认值 | 安全范围 | 影响方面 |
|------|--------|---------|---------|
| 普通出现率 | 60% | 40-80 | 升级期望值 |
| 稀有出现率 | 25% | 15-35 | 升级满意度 |
| 史诗出现率 | 12% | 5-20 | 惊喜感 |
| 传说出现率 | 3% | 1-5 | 极端构建频率 |
| 稀有保底 | 5 次 | 3-8 | 非酋保护 |
| 史诗保底 | 15 次 | 10-20 | 史诗获取下限 |
| 选牌数量 | 3 | 2-4 | 选择空间 |

## 8. 验收标准

- [ ] 升级时显示 3 张随机卡牌，品质颜色正确（白/蓝/紫/橙）
- [ ] 战士只出现战士专属卡和通用卡
- [ ] 法师只出现法师专属卡和通用卡
- [ ] 选择卡牌后效果立即应用到角色属性
- [ ] 同名普通/稀有/史诗卡可重复获得并线性叠加
- [ ] 传说卡获得后不再出现
- [ ] 连续 5 次无稀有时触发保底
- [ ] 连续 15 次无史诗时触发保底
- [ ] 3 张卡牌不出现同名
- [ ] 至少 1 张为当前职业专属卡
