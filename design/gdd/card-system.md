# 卡牌系统 GDD

## 1. 概述

定义升级卡牌的品质、效果、状态效果系统和联动机制。玩家每次升级从 3 张随机卡牌中选择一张，卡牌通过状态效果和触发器系统产生联动组合，让后期构建产生质变而非简单数值堆叠。

## 2. 玩家幻想

前几次升级稳步提升，选择附带了灼烧或冰冻的卡牌。中期开始看到卡牌之间的化学反应——灼烧的敌人被冰冻时瞬间爆发。后期拿到形态变化卡，攻击方式完全改变，从单体输出变成清屏机器。每次升级都让玩家觉得"我变强了"，而不仅仅是"我的数值变高了"。

## 3. 详细规则

### 品质分级

| 品质 | 颜色 | 出现概率 | 定位 |
|------|------|---------|------|
| 普通 | 白色 | 55% | 数值提升 + 小型触发器 |
| 稀有 | 蓝色 | 28% | 状态效果附加 + 中型触发器 |
| 史诗 | 紫色 | 13% | 攻击形态变化 + 状态联动 |
| 传说 | 橙色 | 4% | 机制质变，重新定义玩法 |

### 卡牌类别

| 类别 | 说明 | 出现品质 |
|------|------|---------|
| 数值强化 | 纯数值提升 | 普通 |
| 状态施加 | 攻击附带状态效果 | 普通、稀有 |
| 触发器 | 条件触发型效果 | 稀有、史诗 |
| 形态变化 | 改变攻击方式 | 史诗 |
| 联动 | 消耗/利用状态触发额外效果 | 史诗、传说 |

### 状态效果系统

4 种施加给敌人的 debuff，是卡牌联动的"接口"：

| 状态 | 效果 | 持续时间 | 叠加上限 | 叠加规则 |
|------|------|---------|---------|---------|
| 灼烧 | 每秒造成 5 × 层数的伤害 | 3 秒，每次命中刷新 | 5 层 | 线性叠加 |
| 冰冻 | 移动速度降低 40% | 2 秒 | 不叠加，刷新时间 | 取代灼烧（触发联动） |
| 流血 | 受到所有伤害增加 15% × 层数 | 4 秒，每次命中刷新 | 3 层 | 线性叠加 |
| 眩晕 | 无法移动和攻击 | 0.5 秒 | 不叠加 | 不与其他状态冲突 |

#### 状态交互
- **冰冻取代灼烧**：对灼烧状态的敌人施加冰冻时，冰冻正常生效，灼烧被移除。这个替换过程可以触发联动效果（如"热胀冷缩"）。
- **灼烧 + 流血共存**：两个状态独立运作，灼烧的每秒伤害也会被流血放大。
- **眩晕独立**：眩晕不与其他状态交互，纯粹是控制效果。

### 触发器系统

卡牌可以注册触发器，当特定事件发生时执行额外效果：

| 触发器 | 触发条件 | 设计作用 |
|--------|---------|---------|
| onKill | 玩家击杀敌人 | 奖励进攻，清屏连锁 |
| onHit | 玩家攻击命中敌人 | 高频打击奖励 |
| onDamaged | 玩家受到伤害 | 风险/回报 |
| onLowHp | 玩家 HP 低于 30% | 绝境爆发 |
| onStatusApply | 玩家对敌人施加状态效果 | 联动引擎 |

### 联动组合

3 个核心联动，都是单张卡牌的能力（不需要同时持有两张卡）：

#### 联动 1：热胀冷缩
- **触发**：对灼烧状态的敌人施加冰冻时
- **效果**：瞬间造成灼烧累积总伤害 × 2 的爆发伤害
- **卡牌**："热胀冷缩"（传说，通用）
- **体验**：先远程施加灼烧 → 等累积几层 → 施加冰冻 → 轰！一击清屏

#### 联动 2：连锁反应
- **触发**：击杀敌人时
- **效果**：在击杀位置产生冲击波，对范围内敌人造成当前攻击伤害 50% 的伤害
- **卡牌**："连锁反应"（史诗，通用）
- **体验**：一刀砍死一个 → 冲击波打死旁边两个 → 又触发两个冲击波 → 3 秒清一屏

#### 联动 3：血脉喷张
- **触发**：攻击流血状态的敌人时
- **效果**：每次命中额外造成流血层数 × 5 的伤害
- **卡牌**："血脉喷张"（史诗，通用）
- **体验**：先叠加流血到 3 层 → 之后每次攻击额外 +15 伤害 → 配合高频攻击伤害爆炸

### 选卡流程
1. 升级触发时生成 3 张卡牌
2. 每张卡牌独立按概率抽取品质，再从对应池中随机选择
3. 3 张卡牌不重复（同名不重复出现）
4. 卡牌至少包含 1 张当前职业专属卡
5. 选择后效果立即生效

### 品质保底
- 连续 5 次未出稀有或以上品质，第 6 次保证至少 1 张稀有
- 连续 15 次未出史诗或以上品质，保证至少 1 张史诗
- 传说卡无保底

### 数值叠加
- 同名卡牌可重复获得（除传说卡外）
- 叠加效果为线性累加
- 传说卡每张全局唯一，获得后从池中移除

#### 隐藏联动卡规则

- **解锁条件**：玩家已获得前置组合中的两张卡（同局内）
- **解锁时机**：获得第二张前置卡后，下次升级时隐藏卡进入牌池
- **出现概率**：解锁后按正常品质概率抽取，不额外加权
- **全局唯一**：与传说卡相同，获得后从池中移除
- **与现有联动的互斥**：元素共鸣（共存）与热胀冷缩（替换爆发）代表冰火两条路线，同时持有时按后获得的卡牌规则生效

## 4. 公式

### 品质抽取
```
roll = random(0, 100)
if roll < 4:         → 传说 (4%)
else if roll < 17:   → 史诗 (13%)
else if roll < 45:   → 稀有 (28%)
else:                → 普通 (55%)
```

### 灼烧伤害计算
```
dps = burnDamagePerStack × stackCount × globalDamageMultiplier
totalDamage = dps × duration

变量：
- burnDamagePerStack: 5（基础），可被卡牌强化
- stackCount: 1-5
- duration: 3 秒，每次命中刷新
- globalDamageMultiplier: 1.0（基础）
```

### 流血增伤计算
```
damageMultiplier = 1 + (0.15 × bleedStackCount) + otherBuffs

示例：
- 1 层流血: 伤害 × 1.15
- 3 层流血: 伤害 × 1.45
```

### 热胀冷缩爆发伤害
```
burstDamage = accumulatedBurnDamage × 2

变量：
- accumulatedBurnDamage: 灼烧剩余秒数 × 每秒伤害
- 示例：3 层灼烧还剩 2 秒 = (5×3) × 2 = 30 → 爆发 60 伤害
```

### 连锁反应冲击波
```
shockwaveDamage = currentAttackDamage × 0.5
shockwaveRadius = 60px

连锁：冲击波击杀的敌人也会触发 onKill → 产生新冲击波（无递归保护，最多连锁 5 次）
```

### 血脉喷张额外伤害
```
bonusDamage = bleedStackCount × 5

示例：
- 1 层流血: 额外 +5
- 3 层流血: 额外 +15
```

### 嗜血狂战攻速缩放
```
hpPercent = currentHp / maxHp × 100
fireRateReduction = floor((100 - hpPercent) / 10) × 5%
actualFireRateMs = baseFireRateMs × (1 - fireRateReduction)

示例（战士 baseFireRateMs = 400）：
- 100% HP: 400ms
- 50% HP: 400 × 0.75 = 300ms
- 10% HP: 400 × 0.60 = 240ms
```

### 元素共鸣灼烧伤害
```
if enemy has both 灼烧 and 冰冻:
    burnDps = burnDamagePerStack × stackCount × 2
else:
    burnDps = burnDamagePerStack × stackCount

注意：灼烧与冰冻共存时不触发热胀冷缩的替换爆发
```

### 血之盛宴回复计算
```
onKillBleedingEnemy:
    healAmount = 5 + bleedStackCount × 3

passiveRegen = 2 + bleedingEnemyCount × 1

示例：
- 击杀 2 层流血敌人: 回复 5 + 2×3 = 11 HP
- 场上有 4 个流血敌人: 每秒回复 2 + 4 = 6 HP
```

### 不死狂战减伤缩放
```
hpPercent = currentHp / maxHp × 100
bonusReduction = floor((100 - hpPercent) / 10) × 1
totalReduction = 3 + bonusReduction  // 基础坚韧 3 + 额外

上限：totalReduction ≤ 13（基础 3 + 额外 10）

示例：
- 100% HP: 减伤 3
- 50% HP: 减伤 3 + 5 = 8
- 10% HP: 减伤 3 + 9 = 12
```

### 穿透爆破爆炸计算
```
每穿透一个敌人时在该位置产生爆炸:
explosionRadius = 30px
explosionDamage = bulletDamage × 0.4

一颗穿透 1 次的子弹共产生 2 次爆炸（命中点 + 穿透点）
```

### 弹幕地狱连锁爆破
```
分裂弹也能爆炸:
splitExplosionRadius = 25px
splitExplosionDamage = bulletDamage × 0.3

总效果：命中→爆炸(×0.6)→分裂2颗→各自爆炸(×0.3)
总伤害 = 原始伤害 + 原始×0.6 + 2×原始×0.4 + 2×原始×0.3 = 原始×2.4
```

## 5. 边缘情况

- **热胀冷缩对无灼烧敌人**：不触发，正常施加冰冻
- **连锁反应连锁超过 5 次**：硬上限，第 6 次起不再触发
- **分裂弹 + 穿透弹**：穿透后到达新敌人时分裂，分裂弹不再穿透
- **爆裂弹 + 分裂弹**：命中后先爆炸再分裂，分裂弹不再爆炸（除非持有弹幕地狱）
- **弹幕风暴 + 分裂弹**：3 颗子弹各自独立分裂，共 3+6=9 颗子弹
- **弹幕风暴 + 爆裂弹**：3 颗子弹各自爆炸，共 3 个爆炸
- **弹幕风暴 + 穿透弹**：3 颗子弹各自穿透
- **不死本能触发后 5 秒内再受致命伤**：正常死亡，不死本能已消耗
- **多个 onKill 触发器共存**：各自独立触发，不冲突
- **卡牌池抽空**：传说卡抽完后不再出现，其余品质池足够大
- **元素共鸣 + 热胀冷缩同时持有**：按后获得的卡牌规则生效。后获得元素共鸣→共存生效；后获得热胀冷缩→替换爆发生效。两者互斥，建议只取一条路线
- **隐藏联动卡前置卡被弃**：同局内已获得前置卡才会解锁，跨局不累计
- **穿透爆破爆炸重叠**：多个穿透爆炸可重叠，对同一敌人造成多次伤害（每颗子弹最多命中 1 次）
- **弹幕地狱 + 弹幕风暴**：3 颗子弹各自分裂并爆炸，共 3 个大爆炸 + 6 个小爆炸
- **熔岩战神火焰区域重叠**：多个火焰区域不叠加减速，但灼烧叠加独立计算

## 6. 依赖

### 上游依赖
- **经验与升级**：升级时触发选卡
- **角色系统**：需要职业类型决定专属卡池，攻击参数需数据化
- **战士攻击**：卡牌修改战士攻击参数（attackAngle, attackRange 等）
- **法师攻击**：卡牌修改法师子弹参数（bulletCount, piercing 等）

### 下游依赖
- **状态效果系统**（新增）：卡牌施加的状态效果需要敌人和攻击系统支持
- **HUD**：显示已获得的卡牌列表、当前状态效果
- **敌人系统**：敌人需要支持状态效果（灼烧、冰冻、流血、眩晕）

## 8. 验收标准
