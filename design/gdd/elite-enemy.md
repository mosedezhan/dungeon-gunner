# 精英敌人 GDD

## 1. 概述

基于 5 种普通小兵的强化版本。精英敌人外观与普通版相同但通过发光/颜色特效区分，拥有更高的属性和额外的行为变化，击杀后掉落更多经验。

## 2. 玩家幻想

在普通兵群中突然发现一个发光的精英怪，知道这个家伙更强但奖励也更多。需要快速判断：优先击杀拿经验，还是先清理周围小兵避免腹背受敌。精英怪不是新敌人，而是"更强的老朋友"，玩家一眼就能认出基础类型并调整策略。

## 3. 详细规则

### 通用机制
- 精英敌人基于普通敌人生成，外观相同但有紫色/金色发光特效
- 属性为普通版本的倍率增强
- 击杀后掉落 3 倍经验值
- 出现概率随波次递增

### 精英化规则

| 基础类型 | hp 倍率 | speed 倍率 | contactDamage 倍率 | 精英差异化行为 |
|---------|---------|-----------|-------------------|--------------|
| 普通近战 | ×3 | ×1.2 | ×2 | 攻击半径 +50%，击退效果 |
| 快速兵 | ×2.5 | ×1.3 | ×1.5 | 冲撞产生范围冲击波（半径 40px，伤害 8） |
| 射子弹 | ×3 | ×0.8 | ×1.5 | 每次发射 3 颗子弹（扇形 60°） |
| 重装兵 | ×4 | ×1.0 | ×2.5 | 冲刺距离翻倍（200px），冲刺后地面留减速区域（持续 2s） |
| 爆破兵 | ×2 | ×1.2 | ×2 | 分裂体也是精英属性，爆炸范围 +50% |

### 出现概率
```
eliteChance = 0.05 + waveNumber × 0.02
上限：0.30（30%）

示例：
Wave 1:  7%
Wave 5:  15%
Wave 10: 25%
Wave 13+: 30%（上限）
```

### 生成规则
- 每个敌人生成时独立判定是否为精英
- 判定使用 eliteChance 概率
- 精英敌人也受波次 HP 缩放影响（先应用精英倍率，再应用波次缩放）

### 视觉区分
- 精英敌人周围有紫色发光光环
- 体积略大（radius × 1.15）
- 被击杀时有特殊的碎裂粒子效果

## 4. 公式

### 精英属性计算
```
eliteHp = baseHp × eliteHpMultiplier × (1.15 ^ (waveNumber - unlockWave))
eliteSpeed = baseSpeed × eliteSpeedMultiplier
eliteDamage = baseContactDamage × eliteDamageMultiplier
eliteXpReward = baseXpReward × 3

注意：波次缩放应用在精英倍率之后
```

### 精英出现概率
```
eliteChance = min(0.30, 0.05 + waveNumber × 0.02)

变量定义：
- waveNumber: 当前波次
- 返回值: 每个敌人为精英的概率

范围：
- eliteChance ∈ [0.07, 0.30]
```

### 精英射子弹三连发
```
bulletCount = 3
spreadAngle = 60° (总扇形角度)
每颗子弹角度: -30°, 0°, +30°（相对于玩家方向）
每颗子弹伤害 = 普通子弹伤害 × 1.5
```

### 精英快速兵冲击波
```
冲击波在冲撞终点触发
shockwaveRadius = 40px
shockwaveDamage = 8
shockwaveDuration = 300ms (存在时间)
```

### 精英重装兵减速区域
```
区域半径 = 60px
减速效果 = 移动速度 × 0.5
持续时间 = 2s
不叠加，刷新持续时间
```

## 5. 边缘情况

- **精英爆破兵分裂**：分裂体继承精英属性（hp 按分裂表 × 精英倍率），分裂体也有发光特效
- **精英射子弹子弹**：3 颗子弹独立判定，可分别命中玩家（但无敌帧保护下实际只受 1 次伤害）
- **精英重装兵减速区域重叠**：不叠加，刷新持续时间
- **多个精英同时出现**：各自独立，无特殊联动
- **精英敌人经验球**：掉落 3 个经验球而非 1 个，每个为基础 xpReward 值

## 6. 依赖

### 上游依赖
- **敌人系统**：提供所有普通敌人的基础属性和 AI 行为
- **波次系统**：控制精英出现概率和生成时机

### 下游依赖
- **经验与升级**：精英掉落更多经验球
- **HUD**：可能需要精英击杀计数

## 7. 可调参数

| 参数 | 默认值 | 安全范围 | 影响方面 |
|------|--------|---------|---------|
| 基础精英概率 | 5% | 2-10% | 精英出现频率 |
| 每波递增 | +2% | +1-5% | 后期精英密度 |
| 概率上限 | 30% | 15-50% | 精英数量天花板 |
| HP 倍率 | 2-4× | 1.5-6× | 精英耐打程度 |
| 经验倍率 | 3× | 2-5× | 精英击杀奖励 |
| 发光特效颜色 | 紫色 | — | 视觉识别 |

## 8. 验收标准

- [ ] 精英敌人有紫色发光光环，体积略大于普通版
- [ ] 精英近战兵攻击范围比普通版大 50%
- [ ] 精英射子弹兵每次发射 3 颗扇形子弹
- [ ] 精英快速兵冲撞终点产生冲击波
- [ ] 精英重装兵冲刺距离翻倍，冲刺后留减速区域
- [ ] 精英爆破兵分裂体继承精英属性
- [ ] 精英击杀掉落 3 倍经验值
- [ ] 精英出现概率随波次从 7% 递增至 30% 上限
- [ ] 精英属性先乘精英倍率，再乘波次缩放
