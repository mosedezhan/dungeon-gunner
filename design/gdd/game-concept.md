# 游戏概念文档 — Dungeon Gunner

## 概述

一款 2D 俯视角 Roguelite 射击游戏。玩家选择职业（战士/法师），在无限地图中对抗波次涌来的敌人，通过击杀获取经验升级、收集宝箱获取道具，不断变强直到死亡。每局体验不同——升级卡牌随机、精英敌人随机、Boss 各异。

## 玩家幻想

每次开局都是一次"能走多远"的挑战。操作简单（移动+射击），但策略深度来自职业选择、升级路线、走位决策。快速上手，难以精通。

## 核心循环

```
选择职业 → 对抗波次 → 击杀获取经验 → 升级选卡 → 遇到精英/Boss → 死亡 → 重新开始
```

## 职业

### 战士
- **定位**：近战肉盾
- **攻击方式**：近身 180° 挥砍，朝鼠标方向扇形攻击
- **特点**：高血量、近身高爆发、需要贴脸输出
- **操作**：点击鼠标向鼠标方向挥砍，范围伤害

### 法师
- **定位**：远程输出
- **攻击方式**：远程射击（类似 demo 中的弹幕）
- **特点**：低血量、远程安全距离、无弹药限制
- **操作**：点击鼠标向鼠标方向发射子弹

> MVP 阶段战士+法师足够。

## 敌人体系

### 普通小兵

| 类型 | 血量 | 速度 | 攻击 | 特点 | demo 对应 |
|------|------|------|------|------|----------|
| 普通近战 | 中 | 中 | 近战 | 追踪玩家 | Chaser |
| 快速兵 | 低 | 高 | 近战 | 两点间冲撞 | Rusher |
| 射子弹 | 中 | 慢 | 远程 | 保持距离射击 | Shooter |
| 重装兵 | 高 | 慢 | 高 | 短距离冲刺 | 无（新增） |
| 爆破兵 | 极低 | 极高 | 近战 | 死后分裂 | 无（新增） |

### 精英怪

普通小兵的加强版本，贴图相同，特效/颜色区分，血量和攻击增强，经验掉落增加。

| 基础类型 | 精英差异 |
|----------|---------|
| 普通近战 | 攻击范围变大 |
| 射子弹 | 子弹数量变为3 |
| 重装兵 | 短距离冲刺（接近玩家时触发） |
| 爆破兵 | 死后分裂为2个中等→4个迷你（范围和伤害递减） |
| 快速兵 | 两点之间冲撞 |

### 特殊敌人

**宝箱怪**
- 接近玩家时会逃跑
- 被击杀后掉落宝箱
- 宝箱内容：血量恢复、其他（待定）

**Boss**
- 每人设计一个（待定义）

### 敌人技能 — "泰山压顶"
- 在与玩家较短距离内跳起攻击
- 跳起后"消失"
- 以玩家为目标着陆点
- 降落前提供范围警报（显示落地后的攻击范围）
- 玩家需在落地前躲避
- 暂时搁置，不在 MVP 范围内

## 地图

- 无边界，玩家可自由行走
- 小怪超出当前屏幕一定范围后消失
- 摄像机跟随玩家

> **待讨论**：地图是完全空白还是有地形/障碍物？

## 升级系统

升级时展示 3 张随机卡牌，选择一张。

### 卡牌品质分级
| 品质 | 颜色 | 说明 |
|------|------|------|
| 普通 | 白色 | 基础数值提升 |
| 稀有 | 蓝色 | 较强提升 |
| 史诗 | 紫色 | 强力效果 |
| 传说 | 橙色 | 改变玩法的效果 |

> **待定义**：各品质的具体卡牌列表、出现概率、是否职业差异化

## 经验与升级

- 击杀敌人掉落经验球
- 经验需求公式：`base + level × perLevel`（沿用 demo 公式）
- 升级时弹出选卡界面

## 掉落系统

| 掉落物 | 来源 | 效果 |
|--------|------|------|
| 经验球 | 所有敌人 | 升级 |
| 宝箱 | 宝箱怪 | 血量恢复、待定 |
| 待定 | 精英怪 | 待定 |

## 视觉风格

- 2D 俯视角
- 使用自创美术资源（序列帧雪碧图）
- 运动表现通过序列帧动画实现
- 卡牌效果有视觉冲击（品质颜色区分：白/蓝/紫/橙）
- 精英怪有特效/颜色区分
