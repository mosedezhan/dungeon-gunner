# 移动与输入系统 GDD

## 1. 概述

定义玩家的输入映射、角色移动机制、鼠标瞄准方向和摄像机跟随。玩家通过 WASD 八方向移动，鼠标控制攻击方向，按住鼠标键自动攻击。摄像机平滑跟随玩家，地图无边界。

## 2. 玩家幻想

左手 WASD 走位躲弹幕，右手鼠标瞄准输出方向，操作直觉流畅。移动和攻击方向完全独立——可以边往后退边往前打，经典的"放风筝"体验。

## 3. 详细规则

### 输入映射

| 输入 | 功能 |
|------|------|
| W / 上箭头 | 向上移动 |
| S / 下箭头 | 向下移动 |
| A / 左箭头 | 向左移动 |
| D / 右箭头 | 向右移动 |
| 鼠标位置 | 攻击瞄准方向（角色朝向） |
| 鼠标左键按住 | 自动攻击（按 fireRateMs 间隔连续攻击） |
| ESC | 暂停游戏 |

### 移动机制

- 8 方向移动，对角线速度已归一化（不会斜向跑更快）
- 移动速度由角色系统 moveSpeed 决定
- 松开所有方向键时角色立即停止（无惯性/减速）
- 移动期间角色视觉朝向跟随鼠标方向，不跟随移动方向

### 攻击瞄准

- 角色始终面向鼠标光标方向
- 攻击方向 = 角色中心到鼠标光标的角度
- 战士挥砍扇形以攻击方向为中心
- 法师子弹沿攻击方向直线发射
- 鼠标在角色正上方时朝向 = 0°，顺时针递增

### 自动攻击

- 按住鼠标左键时，按 fireRateMs 间隔自动攻击
- 首次按下时立即触发一次攻击
- 松开鼠标停止攻击
- 攻击期间可自由移动（移动和攻击互不影响）

### 摄像机跟随

- 摄像机以玩家为中心，平滑跟随
- 跟随采用线性插值（lerp），手感顺滑不生硬
- 摄像机偏移：无（玩家始终在屏幕正中央）
- 摄像机不缩放（固定正交视角）

### 地图边界

- 地图无物理边界，玩家可自由移动到任意坐标
- 无可见地图边界标记
- 敌人在屏幕外 200px 处消失（见敌人系统）

### 鼠标光标

- 隐藏系统默认光标
- 使用自定义十字准心光标
- 光标始终显示在游戏画面上

## 4. 公式

### 移动速度
```
inputDirection = normalize(WASD按键组合)
velocity = inputDirection × moveSpeed

示例：
- 只按 W: velocity = (0, -1) × moveSpeed
- 按 W+D: velocity = (0.707, -0.707) × moveSpeed（归一化）
- 无按键: velocity = (0, 0)
```

### 攻击方向
```
attackAngle = atan2(mouseWorldY - playerY, mouseWorldX - playerX)

变量：
- mouseWorldY/X: 鼠标在游戏世界坐标中的位置
- playerY/X: 角色中心坐标
- 返回值: 弧度，0° = 右方，顺时针递增
```

### 摄像机跟随
```
targetCameraX = playerX
targetCameraY = playerY
cameraX = lerp(cameraX, targetCameraX, lerpFactor)
cameraY = lerp(cameraY, targetCameraY, lerpFactor)

lerpFactor = 0.1（每帧，60fps 下约 0.3 秒到达目标）

公式：
lerp(current, target, factor) = current + (target - current) × factor
```

### 自动攻击间隔
```
if mouseDown and (currentTime - lastAttackTime >= fireRateMs):
    executeAttack()
    lastAttackTime = currentTime
```

## 5. 边缘情况

- **WASD 同时按对立方向（W+S / A+D）**：抵消该轴输入，不移动。角色仍面向鼠标
- **鼠标在角色正中心**：攻击方向为上一帧的方向（保持朝向不变）
- **窗口失焦**：释放所有按键状态，暂停游戏
- **鼠标移出游戏窗口**：保持最后已知的鼠标位置，不触发额外攻击
- **按住鼠标时游戏暂停**：恢复后不立即触发攻击，需等待 fireRateMs 间隔
- **极速移动（卡牌大量加移速）**：不设速度上限，但摄像机跟随足够跟住
- **浏览器标签页切换**：Phaser 自动暂停，恢复后时间戳正确处理

## 6. 依赖

### 上游依赖
- **角色系统**：提供 moveSpeed、fireRateMs 等参数

### 下游依赖
- **战士攻击 / 法师攻击**：需要攻击方向（attackAngle）和攻击触发信号
- **敌人系统**：敌人需要玩家位置进行 AI 决策
- **掉落物系统**：拾取判定需要玩家位置
- **HUD**：自定义光标渲染

## 7. 可调参数

| 参数 | 默认值 | 安全范围 | 影响方面 |
|------|--------|---------|---------|
| 摄像机 lerp 因子 | 0.1 | 0.05-0.3 | 跟随手感（越小越平滑） |
| 鼠标方向保持 | 上一帧方向 | — | 鼠标在角色中心的处理 |
| 方向键抵消 | 是 | 是/否 | 对立按键行为 |

- [ ] 角色朝向始终跟随鼠标光标
- [ ] 攻击方向 = 角色到鼠标的角度
- [ ] 按住鼠标左键自动按 fireRateMs 间隔攻击
- [ ] 松开鼠标停止攻击
- [ ] 移动和攻击方向完全独立（可边退边打）
- [ ] 对立方向键同时按下时该轴抵消
- [ ] 摄像机平滑跟随玩家，玩家在屏幕正中央
- [ ] 隐藏系统光标，显示自定义十字准心
- [ ] 窗口失焦时暂停游戏并释放按键状态
