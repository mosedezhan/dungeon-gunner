## Context

游戏文本以硬编码字符串形式分散在各个场景文件和配置文件中。当前没有 i18n 系统，也不需要多语言切换功能。直接替换字符串是最简单直接的方案。

**约束条件：**
- 使用 Courier New 等宽字体，中文字符宽度约为英文的 2 倍，可能导致排版偏移
- 不需要保留英文，直接替换即可
- 翻译风格要求西幻风

## Goals / Non-Goals

**Goals:**
- 所有 UI 文本汉化，用户无需理解英文即可正常游玩
- 升级卡名称/描述采用西幻风格翻译
- 保持 UI 布局不被中文换行破坏（可能需要调整字体大小或容器宽度）

**Non-Goals:**
- 多语言切换系统（不需要英文回退）
- 动态语言加载（一次性静态替换）
- 字体回退机制（Courier New 可以显示中文）

## Decisions

### 1. 直接替换硬编码字符串

**选择**：不引入 i18n 系统，直接在源文件中替换英文字符串为中文。

**理由**：项目不需要多语言切换，引入 i18n 框架是过度设计。直接替换最简单。

### 2. 升级卡翻译风格

**选择**：采用西幻 RPG 风格，简洁有力。

| 原文 | 译文 | 理由 |
|------|------|------|
| Sharper Edge | 锋芒 | 四字风格，有力 |
| Faster Fingers | 迅捷连射 | 描述性翻译 |
| Swift Feet | 疾行 | 简洁 |
| Tough Hide | 坚毅之肤 | D&D 风格 |
| Triple Shot | 三连发 | 游戏术语 |
| Piercing Rounds | 贯穿弹 | 准确描述 |
| Hot Load | 急速装填 | 枪械术语 |
| Regeneration | 愈合 | 简洁 |
| Resonance Core | 共振核心 | 科幻 + 魔法融合 |
| Long Reach | 延伸打击 | 描述性 |
| Wide Sweep | 横扫阔斧 | 成语化 |
| Fervor | 狂热 | 单字有力 |
| Cleaving Edge | 裂地斩 | 西幻技能名 |
| Time Dilation | 时光延展 | 时间系术语 |
| Gravity Well | 引力井 | 科幻术语 |
| Za Warudo | 时停 | 保留 JoJo 简洁感 |

### 3. 保留部分游戏术语

**选择**：
- "WAVE" → 「第 X 波」（完全汉化）
- "HP" → 「生命值」（首次出现时显示完整，HUD 简化为「HP」）
- "XP" → 「经验」（首次出现时显示完整，HUD 简化为「XP」）

**理由**：这些是通用游戏术语，中文玩家已习惯。

### 4. 职业名称/描述

**选择**：
- `mage` — 法师 — 精通冲击波技能
- `warrior` — 战士 — 近战挥砍 / 子弹时间

**理由**：标准翻译，易于理解。

### 5. 字体处理

**选择**：不修改字体配置，依赖浏览器默认的 Courier New 中文字体回退。

**理由**：Courier New 可以显示中文，浏览器会自动使用系统等宽中文字体（如 Consolas + 中文字体组合）。如果排版出现问题，再考虑调整字体大小。

### 6. 场景翻译顺序

**选择**：按用户接触顺序翻译：
1. MenuScene（第一屏）
2. ClassSelectScene（职业选择）
3. GameScene / HUDScene（主界面）
4. UpgradeScene（升级界面）
5. PauseScene（暂停）
6. GameOverScene（结算）
7. DebugScene（调试，最低优先级）

**理由**：按用户游玩路径优先处理高频界面。

## Risks / Trade-offs

### Risk 1: 中文排版破坏

**Mitigation**：中文字符宽度约为英文 2 倍，可能导致：
- 按钮文本溢出
- 卡片描述换行异常
- HUD 统计信息对齐偏移

先完成翻译，在浏览器中测试，必要时调整字体大小或容器宽度。可能需要：
- 减小字体 `fontSize: '16px'` → `'14px'`
- 增加容器宽度
- 调整 `wordWrap` 宽度

### Risk 2: 翻译质量一致性

**Mitigation**：所有翻译在 design.md 中预先定义，确保风格一致。实施时严格对照表执行。

### Risk 3: 遗漏文本

**Mitigation**：通过 `grep` 搜索 `setText`、`text:` 等关键词，确保不遗漏。tasks 中列出每个文件的具体修改项。
